## Elementos estructurales

```html
<!DOCTYPE html>
<!–– Tipo de documento -->
<html>
  <!–– Al inicio y al final -->
  <head>
    <!–– Información para configurar la pagina web -->
    <title></title>
    <!–– Titulo de la web -->
    <meta charset="utf-8" />
    <!–– Codificacion caracteres -->
    <meta name="description" content="Este es un documento HTML5" />
    <!–– Descripción -->
    <meta name="keywords" content="HTML, CSS, JavaScript" />
    <!–– Palabras clave -->
    <link rel="icon" href="imagenes/favicon.png" type="image/png" />
    <!–– Favicon-->
  </head>
  <body>
    <!–– Contenido de la web-->
  </body>
</html>
```

### Elementos del cuerpo

```html
<body>
  <!–– Contenido de la web -->

  <div></div>
  <!-- División genérica -->

  <main></main>
  <!-- División contenido principal -->

  <nav></nav>
  <!-- Navegación -->

  <section></section>
  <!-- Sección genérica -->

  <aside></aside>
  <!--Información relacionada con el contenido principal -->

  <article></article>
  <!-- Artículo independiente -->

  <header></header>
  <!-- Cabecera del cuerpo o de secciones -->

  <footer></footer>
  <!-- Pie del cuerpo o de secciones -->
</body>
```

### Atributos globales

```html
id = "identificador"
<!-- Único -->
class= "identificador"
<!-- Grupo de elementos -->
```
