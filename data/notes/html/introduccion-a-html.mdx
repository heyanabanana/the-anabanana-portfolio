# Introducción a HTML 5
<section>
HTML es un lenguaje de marcado compuesto de elementos que se utilizan para **darles significado** a las partes de una página web.

HTML sirve para indicarle al navegador **qué se muestra** en una página web, pero no debemos usarlo para definir como debe verse ese contenido (si el texto es rojo, si tiene un fontsize predeterminado...). Para ello usaremos CSS.

Estructura básica de un documento .html

`<!DOCTYPE html>` -> Le indicamos al navegador el tipo de documento que va a mostrar.

`<html></html>` -> Iniciamos el documento HTML

`<head></head>` -> Información para el navegador (titulo, descripción, favicon...)

`<body></body>` -> Contenido de la web

```html
<!DOCTYPE html>
<!–– Tipo de documento -->
<html>
  <!–– Al inicio y al final -->
  <head>
    <!–– Información para configurar la pagina web -->
    <title></title>
    <!–– Titulo de la web -->
    <meta charset="utf-8" />
    <!–– Codificacion caracteres -->
    <meta name="description" content="Este es un documento HTML5" />
    <!–– Descripción -->
    <meta name="keywords" content="HTML, CSS, JavaScript" />
    <!–– Palabras clave -->
    <link rel="icon" href="imagenes/favicon.png" type="image/png" />
    <!–– Favicon-->
  </head>
  <body>
    <!–– Contenido de la web-->
  </body>
</html>
```
